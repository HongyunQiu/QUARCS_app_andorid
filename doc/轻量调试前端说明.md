# 轻量调试前端说明

为了减少 APK 构建和安装时的资源体积，现在内嵌前端支持两种模式：

- `full`
- `lite`

## 1. 使用方式

构建轻量调试包：

```bash
cd /home/q/workspace/QUARCS_app_andorid
QUARCS_EMBEDDED_UI_MODE=lite ./build_android.sh
```

一键部署轻量调试包：

```bash
cd /home/q/workspace/QUARCS_app_andorid
QUARCS_EMBEDDED_UI_MODE=lite ./deploy_android.sh
```

默认不设置时，仍然使用完整 stellarium 前端：

```bash
./build_android.sh
```

## 2. 轻量调试页内容

轻量页目录：

`/home/q/workspace/QUARCS_app_andorid/debug-webui`

当前已提供：

- WebSocket 桥接状态显示
- Focuser 串口 / 波特率 / TCP 端点配置
- `FocuserConnect`
- `FocuserDisconnect`
- `FocuserSelfTest`
- `getFocuserParameters`
- `connectAllDevice`
- `disconnectAllDevice`
- `focusMoveStep`
- `focusMoveStop`
- 实时日志面板

## 3. 设计目标

该页面不是产品 UI，而是硬件联调用临时工具页。

核心目标：

- 构建速度快
- 资源体积小
- 进入页面快
- 指令路径短
- 便于直接观察硬件响应
